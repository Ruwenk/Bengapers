# Bengapers

## 🚀 Descripción
Bengapers es un proyecto web con **frontend en React** y **backend en Node.js + Express**, diseñado con una arquitectura modular y escalable.  
Este repositorio contiene tanto el cliente como el servidor, organizados en carpetas separadas.

## 📂 Estructura del proyecto
Bengapers/ ├── frontend/   # Aplicación React ├── backend/    # Servidor Node.js + Express └── README.md   # Documentación principal

---

## ⚙️ Requisitos previos
- [Node.js](https://nodejs.org/) v20+
- [npm](https://www.npmjs.com/) o [yarn](https://yarnpkg.com/)
- Git

---

## 🖥️ Instalación

Clona el repositorio:
```bash
git clone https://github.com/Ruwenk/Bengapers
cd bengapers

frontend 
cd frontend
npm install
npm start

La aplicación se ejecutará en:
👉 http://localhost:3000

Backend
cd backend
npm install
npm run dev
El servidor se ejecutará en:
👉 http://localhost:4000


📌 Scripts útiles
Frontend
- npm start → Ejecuta la app en modo desarrollo
- npm run build → Compila la app para producción
- npm test → Ejecuta pruebas
Backend
- npm run dev → Ejecuta el servidor con nodemon
- npm start → Ejecuta el servidor en modo producción

🛡️ .gitignore
El proyecto incluye un .gitignore para evitar subir:
- node_modules/
- Archivos .env
- Carpeta frontend/build/
- Logs y archivos temporales
- Resultados de pruebas (coverage/, backend/test-output/)
